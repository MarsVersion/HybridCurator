# Hybrid Curator Website

A modern, clean website for Jung Me Chai - Hybrid Curator, built with HTML, CSS, and JavaScript. The site showcases the intersection of curatorial practice and technology, featuring a minimalist design inspired by samgoddard.co.uk.

## 🌟 Features

- **Modern Design**: Clean, minimalist aesthetic with smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Smooth scrolling, hover effects, and mobile navigation
- **Accessibility**: Built with accessibility best practices
- **Performance Optimized**: Fast loading with optimized assets

## 📁 Site Structure

```
/
├── index.html          # Home page
├── about.html          # About page
├── work.html           # Work With Me page
├── projects.html       # Projects showcase
├── journal.html        # Blog/Journal page
├── contact.html        # Contact page
├── styles.css          # Main stylesheet
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## 🚀 Deployment

### Option 1: Static Hosting
1. Upload all files to your web hosting provider
2. Ensure `index.html` is set as the default page
3. Configure your domain to point to the hosting

### Option 2: GitHub Pages
1. Create a new repository
2. Upload all files to the repository
3. Enable GitHub Pages in repository settings
4. Set custom domain if needed

### Option 3: Netlify
1. Drag and drop the project folder to Netlify
2. Configure custom domain
3. Enable form handling for contact form

## 📧 Contact Form Setup

The contact form currently includes client-side validation. For full functionality, you'll need to:

1. **Set up form handling** (e.g., Netlify Forms, Formspree, or custom backend)
2. **Update the form action** in `contact.html`
3. **Configure email notifications**
